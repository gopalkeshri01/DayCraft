# DayCraft

🚀 All-in-one productivity web app to manage your daily routine — includes:

- ✅ To‑Do List  
- 📝 Notes  
- 📅 Daily Planner  
- ⏱️ Pomodoro Timer  
- 💸 Expense Tracker

Built with **React**, **Tailwind CSS**, and **Vite**.

## 🔗 Live Demo

👉 [DayCraft on Netlify](https://daycraftbygopal.netlify.app/)

## 💻 Tech Stack

- React  
- Tailwind CSS  
- Vite  
- LocalStorage for data persistence  
- Lucide-react icons
